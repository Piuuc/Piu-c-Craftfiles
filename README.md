# Piu-c's craftfiles entrepot
This is a place where I store my craft files from various mods incuding Bluedog Design Bureau, Tantares, Tundra Exploration, Coatl Aerospace and more.

I only store the craft files that I think are the most important to me. I did not make every single rocket from every single mods because that's a lot of time and I wont use them all anyway. I only made the most important and useful one in my opinion.

Some craft also include a description, whether it a very basic description of the craft or a more poetic one.
